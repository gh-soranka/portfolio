# Income / Spending Tracker App
- classes created from scretch
- main.py tkinter created with vibe coding / ai research

# Key Features
- Add / Create objects income or spending: with Name, Description, Value and True/False Statement (monthly/yearly)
- Save data in realtime, loading data on start
- Display created objects on screen
- Edit / Delete objects on screen
- Calculation of income/spending values
- Display of calculation results on screen
- Tooltip while mouseover hovering on specific objects / buttons on screen
